evobotgame
==========

An easy to use platform for evolutionary robotics, neuroevolution, and video game AI.

Dependencies:

Python 2.x
Pygame

To run evolution:
-----------------

python botgame.py evolve


To display results:
-------------------

python botgame.py


Note: py-pong adapted from http://code.google.com/p/py-pong/
